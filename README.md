# EasyAndroid
an easy library for developer

使用方法：
compile 'com.easyandroid:easytools:1.2.2'

添加混淆：

-keep class com.easytools.tools.DialogUtil {*;}

-dontwarn com.easytools.tools.**

-keep class com.easytools.constant.**{*;}
